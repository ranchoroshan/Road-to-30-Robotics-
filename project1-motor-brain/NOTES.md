# Motor Brain - C++ Exam Notes

## Key Concepts Learned

### 1. Classes and Objects
- Class is a blueprint
- Object is the actual thing created from blueprint
- Example: Motor is class, m1 is object

### 2. Loops
- for loop = when you know how many times
- while loop = when you run until condition is met

### 3. Functions
- void = function that does action, returns nothing
- Parameters = values you pass into function
- Example: setSpeed(int newSpeed)

### 4. Arrays
- Store multiple values of same type
- Example: Motor motors[4] = four motors
- Access with index: motors[0], motors[1]

### 5. If/Else
- Makes decisions based on conditions
- if / else if / else

### 6. Switch Statement
- Clean way to handle multiple exact values
- Always use break at end of each case

### 7. Pointers and References
- Reference = another name for same variable
- Pointer = stores memory address
- Arrow operator -> used with pointers

## Exam Tips
- Always include #include <iostream>
- Always write using namespace std
- Every opening { needs a closing }
- Class always ends with };
- Main function always returns 0
